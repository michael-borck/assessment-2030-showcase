# Assessment 2030 Showcase

**Presenter:** Michael Borck  
**Event:** Teaching with AI: Showcase presentations for Innovation  
**Organization:** Curtin University

## Overview

This repository contains the presentation materials for the Assessment 2030 Showcase, demonstrating innovative approaches to teaching with artificial intelligence.

## Structure

- `content/` - Quarto markdown files (.qmd) for the presentation content
- `images/` - Supporting images and visual assets
- `docs/` - Rendered static site (GitHub Pages)

## Viewing the Presentation

The presentation is rendered as a static site and available through GitHub Pages. The main presentation content is rendered from the Quarto files in the `content/` directory.

## Development

This project uses Quarto for rendering presentations. To render locally:

```bash
quarto render
```

The rendered output will be placed in the `docs/` directory for GitHub Pages deployment.
